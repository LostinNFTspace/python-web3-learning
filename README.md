# python-web3-learning
Beginner-friendly Python experiments exploring Web3 concepts and Base network interaction.
# iterate-parakeet

## Overview
This repository is part of my learning journey exploring Web3 development using Python,
with a focus on interacting with the Base network.

## What this repo does
- Demonstrates basic Python + Web3 concepts
- Connects to Base RPC
- Reads on-chain data (no transactions)
- Reads ETH balance for a given address on Base

## Why this exists
This is an educational and experimental project intended to:
- Learn Web3 tooling
- Explore the Base ecosystem
- Share simple examples for beginners

## Tech stack
- Python
- web3.py
- Base network
